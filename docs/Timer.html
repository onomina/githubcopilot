_handleComplete()

タイマー終了時の処理。 作業と休憩の間はアラートなしで自動的に次のモードへ切り替わる。 作業＋休憩 = 1 セットとし、全セット（デフォルト 4）完了時のみ onAllSetsComplete を呼び出してタイマーを停止する。 フロー: 1. インターバルを停止 2. 次のモード（作業 ↔ 休憩）と残り秒数を設定 3. 休憩終了なら completedSets を加算 4. 全セット完了かどうかに応じて IDLE / RUNNING に遷移 5. onComplete で UI を更新 6a. 全セット完了 → onAllSetsComplete を呼ぶ（タイマー停止） 6b. 継続中 → _startInterval() で次のモードを自動スタート
