# ポモドーロタイマーアプリケーション 実装機能一覧

## 1. タイマー機能
- タイマーのカウントダウン（分・秒単位で任意の時間設定が可能）
	- 作業時間・休憩時間ともに「分」と「秒」を個別に設定できる
	- 初期値：作業時間「1分00秒」、休憩時間「0分20秒」
- タイマーの開始・一時停止・再開・リセット
- タイマー終了時の通知（アラートや音など）

## 2. UI表示
- 残り時間の表示（分:秒形式）
- 円グラフによる進捗可視化
- 「開始」「リセット」などの操作ボタン
- 現在の状態表示（例：「作業中」「休憩中」など）

## 3. 進捗管理
- 今日のポモドーロ完了回数のカウント
	- 「作業」→「休憩」までを1セットとし、セット完了ごとに1回追加
- 今日の累計集中時間の集計・表示
	- 集中時間は「作業時間」のみを加算していく
- セッション完了時の進捗反映

## 4. 状態管理
- タイマーの状態（作業中／休憩中／停止中など）の管理
- セッションの切り替え（作業→休憩→作業…）
	- 作業終了後は自動で休憩タイマーを開始、休憩終了後は自動で作業タイマーを開始

- 今日の進捗データの一時保存（リロード時の保持など、localStorageやセッション等）
## 5. データ保存
 - 今日の進捗データの一時保存（リロード時の保持など、localStorageやセッション等）

## 6. テスト関連
 - タイマー・進捗管理ロジックのユニットテスト
